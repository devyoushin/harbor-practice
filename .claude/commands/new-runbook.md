새 Harbor 운영 런북을 생성합니다.

**사용법**: `/new-runbook <작업명>`

**예시**: `/new-runbook Harbor 스토리지 확장`

작업 유형:
- `레지스트리 관리`: 프로젝트/로봇 계정 생성
- `복제 설정`: 원격 레지스트리 동기화
- `긴급 대응`: 스토리지 풀, 인증 실패
- `업그레이드`: Harbor 버전 업그레이드

런북 포함 내용:
- 사전 체크리스트 (Harbor 상태, 스토리지 현황)
- Harbor API 또는 kubectl 명령어
- 백업 및 롤백 절차
- 모니터링 포인트 (S3 버킷 사용량, PVC 용량)
