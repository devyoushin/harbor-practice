# 보안 체크리스트 — harbor-practice

## 접근 제어
- [ ] 관리자 기본 비밀번호 변경
- [ ] LDAP/OIDC 연동 (개인 계정 금지)
- [ ] 로봇 계정: 프로젝트 범위로 제한
- [ ] 로봇 계정: 만료 기간 설정 (90일 이하)
- [ ] 프로젝트 접근 레벨: Private (Public 최소화)

## 이미지 보안
- [ ] 취약점 스캐너 활성화 (Trivy 연동)
- [ ] Push 시 자동 스캔 설정
- [ ] Critical 취약점 이미지 배포 차단 정책
- [ ] 이미지 서명 검증 활성화 (Cosign)
- [ ] 정기 스캔 스케줄 설정 (주 1회)

## 스토리지 보안
- [ ] S3 버킷: 퍼블릭 접근 차단
- [ ] S3 버킷: 서버 사이드 암호화 (SSE-S3 또는 SSE-KMS)
- [ ] IRSA로 S3 접근 (액세스 키 하드코딩 금지)
- [ ] PVC 암호화 활성화

## TLS/네트워크
